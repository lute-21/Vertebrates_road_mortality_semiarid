# Vertebrates_road_mortality_semiarid
 Raw and Processed Datas about road-kills vertebrates

#Repositório criado por Luiza Teixeira/ doutoranda do PPG Ecologia -UFRGS

#Este respositório possui dados de atropelamento de vertebrados terrestres da BR-122 no município de Quixadá-Ce do período de 2017 a 2018.

#A pasta denominada denominada Data estão os dados brutos  e os dados processados por trecho e por mês de amostragem.

	#Na pasta Doc há o arquivo de texto que foi submetido a 	revisata Austral Ecology
	
	#Na pasta output você irá encontrar, figuras, tabelas e 	arquivos klm, com os resultados de hotspots da 	mortalidade de vertebrados
	
	#Na pasta R há todos os scripts utilizados para carregar 	os arquivos, juntamente com os scripts com análise
	
	#Finalmente na pasta Siriema estão os dados organizados 	com suas respectivas coordenadas para utilização do 	programa Siriema que pode ser baixado no link. Com ele é 	possível rodar os dados utilizando a R-ripley, para 	apresentar onde ocorreram hotspots.
